自己破産とは？福井の弁護士がわかりやすく解説

「自己破産」という言葉に不安を感じていませんか？
正しく理解すれば、人生を立て直すための有効な手段です。

自己破産は、法律で認められた「やり直し」の制度です

自己破産とは、借金の返済が困難になった方が、裁判所に申立てを行い、免責（支払義務の免除）を受ける手続きです。破産法に基づき裁判所で行われる、生活を再建するための法的な仕組みです。

「自己破産」と聞くと、「人生が終わる」「すべてを失う」というイメージをお持ちの方が多いかもしれません。しかし、実際にはそのようなことはありません。自己破産は、経済的に行き詰まった方が新たなスタートを切るために用意された制度です。

自己破産をしても、生活に必要な家財道具や一定額の現金は手元に残すことができます。戸籍に記載されることもありません。選挙権がなくなることもありません。多くの方が自己破産後に安定した生活を取り戻しています。

自己破産と「免責」の関係

自己破産の手続きには、「破産手続」と「免責手続」の2つのステップがあります。

破産手続は、債務者の財産を調査・換価（お金に換えること）して、債権者に公平に分配する手続きです。

免責手続は、破産手続で返しきれなかった借金について、裁判所が免責許可決定を出して支払義務を免除する手続きです。

つまり、自己破産の最大の目的は「免責」を得ることです。免責許可決定が出れば、原則としてすべての借金の返済義務がなくなります。実務上、自己破産を申し立てるほとんどの方が免責許可を受けています。

なお、税金や養育費など一部免責されない債務もあります。詳しくは弁護士にご確認ください。

自己破産が向いている方

以下のような状況の方は、自己破産を検討する価値があります。

✓ 借金の総額が大きく、返済の見通しが立たない
✓ 収入がない、または収入が少なく返済ができない
✓ 複数の貸金業者から借入があり、返済が追いつかない
✓ 毎月の返済のために別のところから借りている
✓ 督促の電話や手紙が届いて精神的に追い詰められている
✓ 病気やケガで働けなくなり、返済ができない
