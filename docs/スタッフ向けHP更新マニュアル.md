## このマニュアルについて

債務整理HPの文字修正・軽微な更新を、スタッフがブラウザ上で行うための手順書です。
特別なソフトのインストールは不要です。ブラウザ（Chrome等）だけで作業できます。

### 大事なルール（必ず守ること）

1. **触っていいのは「日本語のテキスト」と「数字」だけ**
2. **`< >` で囲まれた部分（HTMLタグ）は絶対に変更しない**
3. **不安な場合は、変更前に代表に確認する**
4. **間違えても元に戻せるので、慌てなくて大丈夫です**

## 3. ファイルを編集する

### 3-1. 編集モードに入る

ファイルを開くと、コードが表示されます。

画面の右上あたりにある **鉛筆マーク（✏）** をクリックします。
（ボタンにカーソルを合わせると「Edit this file」と表示されます）

すると、テキストが編集できる状態になります。

### 3-2. 修正したい箇所を探す

**`Ctrl + F`**（キーボードのCtrlキーを押しながらFキー）を押すと、検索窓が出ます。

修正したいテキスト（例：「0776-28-2824」や「初回相談無料」など）を入力すると、その箇所がハイライトされます。

### 3-3. テキストを修正する

**修正していいのは、日本語のテキストと数字だけです。**

#### OK（修正してよい例）

```
変更前：<p>予約受付：9:30〜20:00</p>
変更後：<p>予約受付：9:30〜19:00</p>
        ↑ 数字だけを変更。<p>と</p>はそのまま
```

```
変更前：<td>385,000円〜</td>
変更後：<td>400,000円〜</td>
        ↑ 金額だけを変更。<td>と</td>はそのまま
```

```
変更前：<span>毎月の返済が苦しく、生活が回らない</span>
変更後：<span>毎月の返済が苦しく、生活費が足りない</span>
        ↑ 日本語だけを変更。<span>と</span>はそのまま
```

#### NG（触ってはいけない例）

```
× <p>を消してしまう → 表示が崩れます
× <a href="tel:0776-28-2824"> の href="" の中を消す → リンクが壊れます
× class="section__title" を変更する → デザインが崩れます
```

**迷ったら、変更せずに代表に確認してください。**

## 4. 変更を保存する（コミット）

### 4-1. 保存画面を開く

編集が終わったら、画面右上の **「Commit changes...」** という緑のボタンをクリックします。

### 4-2. 変更内容のメモを書く

ポップアップ画面が出ます。

| 項目 | 入力内容 |
|------|---------|
| **Commit message**（1行目） | 何を変更したか短く書く（例：「電話番号を変更」「費用の金額を更新」） |
| **Extended description**（2行目以降） | 空欄でOK |

### 4-3. 保存を実行する

**「Commit changes」** ボタンを押します。

これで変更が保存されました。**数分後にHPに自動で反映されます。**

**反映されない場合は、5分ほど待ってから再度確認してください。**

## よくある質問

### Q. HTMLタグって何ですか？

`< >` で囲まれた部分のことです。例えば `<p>` `</p>` `<h2>` `<a href="...">` など。
これらはページの構造を定義する記号で、変更するとページの表示が壊れます。
**日本語や数字以外は触らない** と覚えておけばOKです。

### Q. 画像を差し替えたい場合は？

画像の差し替えは代表に依頼してください。Claude Codeで対応します。

### Q. 新しいページを追加したい場合は？

新規ページの作成は代表に依頼してください。Claude Codeで対応します。

### Q. 複数のファイルを同時に修正したい場合は？

1ファイルずつ修正してください。手順3〜4を繰り返せばOKです。

### Q. スマホからでも編集できますか？

できますが、画面が小さく操作しづらいため、パソコンでの作業を推奨します。
